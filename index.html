Internet - Where people can communicate on their computers/mobile devices/tablets by accessing web pages.
The http is a request sent to a server that is holding that web address. The server then sends back
HTML (Hyper Text Markup Language) which the web browser (Chrome, Safari, Firefox, etc.) then interprets
for the end user. The majority of the internet is made up of HTML, which is the contect we see.

How we see the web pages is based partially on the tags and elements that a programmer writes so that the computer
understands. Tags have an opening tag and a closing tag and in between is the content. When you look at the whole
tag including its content it is called an element. An example of a tag would be to make a font bold or italics.
There are two typles of HTML elements, inline and block. Still a little fuzzy on the reason why you would use block,
but I'm sure it will be made clear soon enough. I do now that block forms an invisible box and inline is a
line at a time.
